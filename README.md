# SolPredict — Molecular Solubility Prediction

Predict aqueous solubility of organic molecules using Random Forest and Neural Network
models trained on the [ESOL dataset](https://pubs.acs.org/doi/10.1021/ci034243x).

## Overview

- **ML Pipeline:** RDKit for molecular fingerprints, scikit-learn RF + PyTorch MLP
- **API:** FastAPI on HuggingFace Spaces
- **Dashboard:** Next.js on Vercel with Recharts visualizations

## Quick Start

### Training

```bash
pip install -r requirements.txt
python scripts/train.py
```

### API (local)

```bash
uvicorn api.main:app --port 7860
```

### Dashboard (local)

```bash
cd web && npm install && npm run dev
```

## Project Structure

- `solpredict/` — Shared Python package (featurization, model definition)
- `scripts/train.py` — Training pipeline
- `notebooks/training.ipynb` — Research notebook with full analysis
- `api/` — FastAPI prediction service
- `web/` — Next.js dashboard
- `data/` — ESOL dataset and exported results
- `models/` — Trained model artifacts (generated by train.py)

## Results

See `data/results.json` for full metrics, or visit the live dashboard's Model Comparison page.

## Deployment

### Vercel (Frontend)
1. Connect GitHub repo
2. Set root directory: `web/`
3. Add env var: `NEXT_PUBLIC_API_URL=https://your-hf-space.hf.space`

### HuggingFace Spaces (API)
1. Create a new Docker Space
2. Upload: `api/`, `solpredict/`, `models/`
3. The Dockerfile handles the rest
