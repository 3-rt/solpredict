# SolPredict — Molecular Solubility Prediction

[![CI](https://github.com/3-rt/solpredict/actions/workflows/ci.yml/badge.svg)](https://github.com/3-rt/solpredict/actions/workflows/ci.yml)

Predict aqueous solubility of organic molecules using Random Forest and Neural Network
models trained on the [ESOL dataset](https://pubs.acs.org/doi/10.1021/ci034243x).

## Overview

- **ML Pipeline:** RDKit for molecular fingerprints, scikit-learn RF + PyTorch MLP
- **API:** FastAPI on HuggingFace Spaces
- **Dashboard:** Next.js on Vercel with Recharts visualizations

## Quick Start

You need **two terminals** to run the full stack locally (API + dashboard).
Training is a one-time setup step you run first.

### 1. Install & Train (one-time setup)

All commands assume you're in the **project root** (`solpredict/`).

```bash
# Create a virtual environment and install dependencies
python3 -m venv .venv
source .venv/bin/activate
pip install -e ".[dev,test,api,notebook]"

# Train the models (generates files in models/)
python scripts/train.py
```

### 2. Start the API — Terminal 1

```bash
# From the project root, with the venv activated
# (the install from step 1 already includes the API deps via the [api] extra)
source .venv/bin/activate
uvicorn api.main:app --port 7860
```

The API will be running at `http://localhost:7860`. Keep this terminal open.

### 3. Start the Dashboard — Terminal 2

```bash
# From the project root, in a new terminal
cd web
npm install
npm run dev
```

The dashboard will be running at `http://localhost:3000`. It talks to the API on port 7860.

## Project Structure

- `solpredict/` — Shared Python package (featurization, model definition)
- `scripts/train.py` — Training pipeline
- `notebooks/training.ipynb` — Research notebook with full analysis
- `api/` — FastAPI prediction service
- `web/` — Next.js dashboard
- `data/` — ESOL dataset and exported results
- `models/` — Trained model artifacts (generated by train.py)

## Development

```bash
# Install all dev + test dependencies (already done if you ran step 1)
pip install -e ".[dev,test,api,notebook]"

# Install pre-commit hooks
pre-commit install

# Run the full quality suite
ruff check solpredict tests
ruff format --check solpredict tests
mypy solpredict
pytest
```

Configuration lives in `pyproject.toml` (project metadata, deps, ruff, mypy, pytest, coverage).
